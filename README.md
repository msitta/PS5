# PS5
Tripleten DA Projeto Sprint 5
O app serve para criar dois tipos de gráfico que representam os carros usados à venda nos EUA:
O histograma mostra a quantidade de carros por faixas de quilometragem rodada em intervalos de 2.000 km;
O gráfico de dispersão traz como dados a relação entre preço e quilometragem rodada, com dados individuais de cada veículo.

O link para o repositório no Github é "https://github.com/msitta/PS5.git"

O link para o resultado desse projeto é "https://projeto-da-sprint-5.onrender.com/"
